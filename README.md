# 🦊 Cruzeiro - Acervo Histórico

> **"A Imagem Resplandece"** — Uma landing page premium dedicada à gloriosa história do Cruzeiro Esporte Clube.

Este projeto é uma exposição virtual que combina design sofisticado com a trajetória monumental do Maior de Minas, apresentando desde as "Glórias Eternas" na sala de troféus até os "Heróis Celestes" que fizeram história nos gramados.

##  Demonstração

O projeto está publicado e pode ser visualizado aqui:
**[Aceder ao Deploy](https://zezo-00.github.io/Landing-page-Cruzeiro/)**

## Funcionalidades e Destaques

* **Design Pixel Perfect:** Interface inspirada em designs modernos de alta fidelidade, focada em tipografia elegante e espaçamento harmônico.
* **Efeitos Visuais Avançados:** Uso de degradês dinâmicos em títulos estratégicos e números de conquistas.
* **Galeria de Lendas:** Secção organizada em *Z-Pattern* (intercalada) com foco em imagens de alta resolução e biografia dos ídolos.
* **Componentização:** Código totalmente modularizado utilizando React.
* **Performance:** Desenvolvido com Vite.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

* **[React](https://reactjs.org/):** Biblioteca principal para a construção da interface.
* **[Vite](https://vitejs.dev/):** Build tool de próxima geração para um desenvolvimento rápido.
* **[TypeScript](https://www.typescriptlang.org/):** Para garantir tipagem estática e maior segurança ao código.
* **CSS3 Moderno:** Flexbox, Grid, Variáveis CSS e efeitos de degradê de texto.
* **Google Fonts:** Utilização das fontes *Playfair Display* (Serif) e *Inter* (Sans-serif).

## Como Executar o Projeto Localmente

Se desejar clonar o projeto e rodar na sua máquina, siga os passos:

1. **Clonar o repositório:**
```bash
git clone https://github.com/zezo-00/Landing-page-Cruzeiro.git

```


2. **Entrar na pasta:**
```bash
cd Landing-page-Cruzeiro

```


3. **Instalar as dependências:**
```bash
npm install

```


4. **Iniciar o servidor de desenvolvimento:**
```bash
npm run dev

```


5. Abra o navegador em `http://localhost:5173`.
